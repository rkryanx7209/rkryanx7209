## 📊 Destaque: Data-Social SESI SENAI

> Plataforma de visualização de dados socioeconômicos que consome a API oficial do IBGE/SEADE, apresentando informações sobre Brasil, São Paulo e as 39 cidades do Vale do Paraíba.

### Funcionalidades:
- 📈 Dashboard interativo com drag-and-drop
- 🗺️ Mapa interativo das cidades do Vale
- 📊 Gráficos dinâmicos (ApexCharts)
- 🔄 Comparação entre regiões
- 💾 Persistência local dos dashboards

### Tecnologias utilizadas:
`PHP` • `JavaScript` • `ApexCharts` • `Leaflet` • `Bootstrap Icons` • `IBGE API` • `SEADE API`
